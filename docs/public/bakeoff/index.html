CSS 3D

0 KB dep · real DOM

Crisp + accessible + auto-themed. Real text (selectable, screen-reader-able), zero dependency, tracks the palette for free. No true lighting; for a scripted scroll path that costs nothing.

OGL

~16 KB gz · lean WebGL

Real 3D, tiny. True perspective at 1/8th of Three’s weight. Flat-shaded here (you add lighting yourself); text is a canvas texture, so slightly softer than DOM. Smaller ecosystem to lean on.

Three.js

~131 KB gz · WebGL + lighting

Richest + most popular. Real lighting, huge ecosystem, easy future free-orbit. Heaviest (lazy-loaded), needs an a11y / no-WebGL fallback. Text is a canvas texture.
