You have been on the receiving end of slide decks where every slide is
slightly different from the one before. The headline is in one font on
page 4 and another on page 5. The accent blue drifts between two shades.
A chart copied from a 2022 deck still says "FY22" in the corner. Nobody
planned the drift — it happens because slide tools make every slide a
blank canvas, and humans are humans.

**Lattice is what you build when you want to stop that from happening.**
The author writes the deck as a plain Markdown file, picks one of the
named layouts for each slide, and the engine assembles the deck against a
shared palette. Every deck looks like it came from the same team —
because, structurally, it did.
