Frame → Cell

A Frame is a named set of Cells and how they're arranged on the slide. Different frames pick different cells — that's all a frame is in 2D.

stage

Read it as: the frame's manifest is just a cells:[…] list (+ suppresses:[…]). standard composes nine cells; split-panel keeps only stage and suppresses the chrome. The frame owns the relationship between cells — a stack on the rectangle in 2D; the 3-D carving in a spatial renderer. It's recursive: a Cell holds a Tile or another Frame — `split-panel` is a sovereign Frame docked in the stage Cell.
