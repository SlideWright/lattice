# Five killer word cloud layouts.

A real spiral packer. Multi-hue palette. Continuous weights. Boardroom-ready.

---

<!-- _class: subtopic -->

## What this deck shows.

Five word-cloud variants from one source contract. Author writes a list of terms with a trailing weight `1`–`5` (continuous floats allowed); a build-time Archimedean spiral packer places each word around the canvas centre, sizes it continuously from the weight, rotates a deterministic share 90°, and cycles colour through six categorical tokens. One default plus four modifiers — `constellation`, `dense`, `spectrum`, `focal` — that each rebind the engine parameters (density, rotation policy, colour mode, size spread). Zero client-side randomness, zero dependencies, palette-blind.
