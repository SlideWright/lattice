# Two categorical systems, two clear names.

`Universal token system · Phase 6`

*The chart spectrum moves off the bare `--cat1-*` (which sat a hyphen from phase 1's diagram `--cat-1-*`) onto its own `--chart-cat-*`.*

---

<!-- _class: cards-grid -->

## The near-collision, resolved.

- The problem
  - Phase 1 introduced `--cat-1-fill` (diagram, 12 slots). The chart spectrum was `--cat1-fill` (8 slots) — one hyphen apart, easy to confuse.
- Diagram categorical
  - `--cat-1-fill` / `--cat-1-mark` — the 12-slot band cycle for Mermaid diagrams.
- Chart categorical
  - `--chart-cat-1-hue` / `-fill` / `-ink` — the 8-slot Apple-tuned spectrum for charts.
- Kept distinct
  - Two systems by design (Wong 2011: charts cap at ~8 hues). Now the names say which is which.
