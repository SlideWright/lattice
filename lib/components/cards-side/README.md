# cards-side

> Two cards side-by-side, co-equal.

**Function** inventory · **Form** split · **Substance** structure

## When to use

Use for an explicit pair — two options, two phases, two artifacts presented with equal weight.

## Authoring

See `example.md` for the canonical authoring shape.

|---|---|---|---|
| title | `h2` | yes | Slide heading. |
| cards | `ul > li` | yes | Exactly two list items, each one card. Lead each with **Card Title.** then body text. |
